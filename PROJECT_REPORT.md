Introduction  
User success and long-term retention are critical for the growth of any SaaS platform. However, identifying early behavioral patterns that indicate whether a user will succeed or disengage remains a challenge. This project focuses on analyzing user behavior data to uncover meaningful signals that differentiate successful users from at-risk users.

The project was developed as part of the Zerve Hackathon with the goal of applying practical analytics and machine learning techniques to real-world user success problems.

Exploratory Analysis

Behavior-based features were engineered to better represent user intent and engagement, including:
